A short 2-hour creation that visualizes numerous IMDb movie entries.
Each type of emoji represents a different genre; see the legend.
Hover over an emoji to see the represented movie and basic information.
Click on the emoji to see the IMDb site for the movie.
X-axis location represents release year; y-axis location represents rating #/10.
Zoom with scroll, pan with click and drag.

Keep in mind, you may have to download the dataset CSV file (the last cell in the Colab).
Then, add it to the Colab workspace files.
